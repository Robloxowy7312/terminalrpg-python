## Description

The game currently has no objective, as I'm still working on it. Still, you can:
1. Save your progress
2. Load your progress
3. Permanently die (removes save file)
4. Fight monsters
5. Walk around


## Requirements

The project requires [Python 3.12.3](https://www.python.org/downloads/release/python-3123/) or later and the `rich` library for Python.

## Installation

1. Install Python by following the steps used for your OS.
2. Install the `rich` library:
```bash
pip install rich
```
3. Unzip the terminalrpg-python-main.zip file. 
4. Run the main.py file by either double-clicking it or by opening the folder in your terminal and executing
```bash
python3 main.py
```
in it.

## Usage
The game controls are explained in-game. The game should be compatible with all desktop operating systems.
